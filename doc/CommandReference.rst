=================
Command Reference
=================

The daisy framework includes a variety of commands and commands. These
tools are available through the :file:`daisy` command line
interface. To list all available tools, type::

   daisy

To run a particular tools, type::

   daisy <toolname>

for example, to get help on the `run` tool, type::

   daisy run --help
